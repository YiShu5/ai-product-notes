# C语言新手小白详细教程（7）指针和指针变量

> 原文：[C语言新手小白详细教程（7）指针和指针变量](https://blog.csdn.net/2302_79751907/article/details/141070568)  
> 作者：意疏｜许可：CC BY-SA 4.0

指针是存储内存地址的变量。理解指针的关键，是区分“变量的值”和“变量所在的位置”。

```c
int number = 10;
int *p = &number;

printf("%d\n", number);  // 10
printf("%d\n", *p);      // 10
```

- `&number`：取变量 `number` 的地址。
- `int *p`：声明一个指向 `int` 的指针变量。
- `*p`：解引用，访问 `p` 指向地址中存放的值。

通过指针可以间接修改原变量：

```c
int number = 10;
int *p = &number;
*p = 20;
printf("%d\n", number);  // 20
```

指针类型决定了解引用时按什么类型解释内存，也会影响指针运算的步长。初学阶段应避免使用未初始化的指针；让指针指向一个有效对象，或将其初始化为 `NULL`。
