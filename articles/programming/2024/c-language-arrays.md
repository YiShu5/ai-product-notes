数组用于存放一组相同类型的数据。下标从 `0` 开始，访问时必须保证不越界。

## 一维数组

```c
int scores[5] = {90, 82, 76, 95, 88};
printf("第一个分数：%d\n", scores[0]);
```

- `int scores[5];`：声明包含 5 个 `int` 元素的数组。
- `scores[i]`：访问下标为 `i` 的元素。
- 未显式初始化的局部数组元素值不确定；应在使用前初始化。

```c
int a[5] = {1, 2};  // 其余元素自动补 0
```

## 二维数组

二维数组可以理解为“数组中的数组”，常用于表格、矩阵等场景。

```c
int table[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
printf("%d\n", table[1][2]); // 6
```

遍历二维数组通常使用两层循环：外层控制行，内层控制列。

```c
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        printf("%d ", table[i][j]);
    }
    printf("\n");
}
```
