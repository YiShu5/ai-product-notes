# C语言新手小白详细教程（8）ASCll编码和字符串

> 原文：[C语言新手小白详细教程（8）ASCll编码和字符串](https://blog.csdn.net/2302_79751907/article/details/141942241)  
> 作者：意疏｜许可：CC BY-SA 4.0

字符在计算机中以数值编码保存。常见的 ASCII 编码中，`'A'`、`'a'`、`'0'` 都对应各自的整数值；在 C 语言中，`char` 用来保存单个字符。

```c
char ch = 'A';
printf("%c %d\n", ch, ch);
```

字符串则是一串字符，并以空字符 `\0` 作为结束标记。

```c
char name[] = "Yishu";
printf("%s\n", name);
```

`"Yishu"` 在内存中实际包含 `Y`、`i`、`s`、`h`、`u` 和结尾的 `\0`。因此声明字符串数组时要为结束符留出空间：

```c
char word[6] = "hello";
```

处理字符串时，应注意数组容量，避免写入超过数组边界的数据。
