## 前言

最近，一款新兴人工智能 DeepSeek 火爆出圈，众多技术大佬纷纷投身其本地部署操作，关键是部署后离线也能随心使用。毫不夸张地说，DeepSeek 的实力与 OpenAI 的 GPT 相比也毫不逊色。接下来，我将一步一步详细地为大家讲解如何在本地部署 DeepSeek，带你解锁这一强大工具的便捷使用方式 。

### 安装ollama

在开始本地部署 DeepSeek 之前，需要先安装 ollama。 这是ollama的安装地址ollama 直接点击下载即可。下载过程中，强烈推荐使用谷歌浏览器进行操作，要是你具备科学上网条件，建议开启，这样能有效避免下载时可能出现的各类网络问题。

我们打开上面的网址之后 点击红色框里面的这个 我们选择适合自己系统的下载就好了 我们静静等待下载(注意下载是需要魔法的) 这是我们下载好的文件，双击打开它 我们点击下载 等待安装… 我们安装好之后 是看不到任何的图形界面的 我们点击桌面右下角 如果出现羊驼图标 就代表安装成功了 还有一种方法可以判断ollama是否安装成功，就是win + r输入cmd调出命令行 进入命令行模式。 在命令行中输入 ollama -v，然后按下回车键。如果屏幕上出现了 ollama 的版本号，就意味着你已经成功安装 ollama。这一步直接输入ollama也是可以的

- 注：这里我们的命令行界面不要关闭 等下拉取模型的话可以使用到

### ollama拉取DeepSeek R1模型

我们首先点击Modles 之后点击deepseek-r1 我们这里选择部署14b的 点击14b 之后点击黄色方框的复制。 我们点击这个之后 输入ollama run deepseek-r1:14b

如果下载很慢的话 可以ctrl + c重新下载就会变快的 这里出现success就代表下载完成了 我们可以使用ollama list命令来查看拉取的大模型 我们查看完模型名字之后 可以使用ollama run deepseek-r1:14b命令来运行deepseek 运行之后 我们就可以问问题了 这里我问了如何打开csdn网站
