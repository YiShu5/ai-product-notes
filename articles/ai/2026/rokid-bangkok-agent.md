# 我给 Rokid 眼镜装了个泰酷辣，然后带它去了曼谷

> 原文：[https://blog.csdn.net/2302_79751907/article/details/158816464](https://blog.csdn.net/2302_79751907/article/details/158816464)  
> 作者：意疏｜许可：CC BY-SA 4.0

我承认，出发前我想过很多可能翻车的场景：

夜市摊位上全是泰文的菜单、突突车司机用泰语报了个价你完全听不出数字、7-Eleven 店员语速飞快地问你"要不要袋子"——

这些瞬间，掏手机打开翻译 App 也能应付。但你试过一边拖着行李箱一边举手机对着路牌拍照翻译吗？那个画面，狼狈得像在街头玩体感游戏。

这次曼谷行，我全程戴着 Rokid 智能眼镜，用灵珠 AI 平台搭了一个"泰酷辣·AI 随身搭子"智能体。

五天下来，最大的感受只有一句话：

你终于可以像本地人一样走在街上了——抬着头，不低头。

### 1️⃣ 先说体感：翻译这件事，为什么非得用眼镜？

手机翻译能用吗？能用。Google Translate、DeepL 都很强。

但问题出在交互链路上：

手机翻译的动线： 听到泰语 → 掏手机 → 解锁 → 打开 App → 点麦克风 → 等结果 → 看屏幕 → 抬头回应对方。

Rokid 的动线： 听到泰语 → 镜片上出现中文 → 直接回应对方。

中间差了六步操作和三秒尴尬。

三秒够干嘛？够对面那个泰国摊主把"萨瓦迪卡"的笑容从热情变成敷衍。

对话这件事，讲的就是眼神和节奏。你一低头看手机，那个"人和人在聊天"的感觉就碎了。Rokid 让翻译这件事退到了背景里——你看着对方的眼睛，字幕在你们之间。

翻译最好的状态，是你忘了你们说的不是同一种语言。

### 2️⃣ 灵珠 AI 智能体：不只是翻译，是半个当地朋友

灵珠 AI 平台上搭智能体，门槛比我想象的低。

再从魔搭社区 获取mcp URL 就可以了

我给这个智能体起名叫"泰酷辣·AI 随身搭子"，喂了三类知识：

- 泰国旅行常识（BTS/MRT 怎么坐、突突车怎么砍价、7-Eleven 能干什么）
- 餐饮文化知识（夜市点餐技巧、路边摊辣度分级、"ไม่เผ็ด"是什么意思）
- 应急话术库（药店买药怎么描述症状、坐船坐过站了怎么问、酒店 check-in 常用对话）

它能做到的事情举几个例子：

你说： “给我翻译屏幕上的泰文？”

你问智能体 ： 泰国秤上作弊怎么处罚

翻译软件给你答案，智能体给你判断。一个是词典，一个是朋友。

### 3️⃣ 技术链路：从声音到镜片，200 毫秒

89 种语言支持，中英日德法西 6 种离线可用。

就算你在曼谷 BTS 地下段没有信号，泰语翻译照样跑。

整个翻译链路拆开来看：

```
麦克风拾音 → VAD 端点检测（判断一句话说完没有）
 → ASR 语音识别（泰语 → 泰文文本）
 → NMT 神经机器翻译（泰文 → 中文）
 → 灵珠AI语境优化（结合场景微调译文）
 → CXR-M SDK 推送至眼镜端
 → 镜片 UI 渲染字幕
```

端到端延迟 200-300ms。人类对话的自然停顿大约 400-600ms，所以翻译结果几乎是"对方话音刚落，中文就出来了"。

核心推送逻辑只有几行：

```
// 打开翻译场景
CxrApi.getInstance().controlScene(ValueUtil.CxrSceneType.Translation, true, null)

// 配置显示参数
CxrApi.getInstance().configTranslationText(22f, 60, 180, 880, 400)

// 推送翻译结果（vadId 必须递增，否则会覆盖）
vadId++
CxrApi.getInstance().sendTranslationContent(vadId, 0, false, true, "$original\n$translated")

// 退出时关闭场景
CxrApi.getInstance().controlScene(ValueUtil.CxrSceneType.Translation, false, null)
```

### 4️⃣ 导航：视线里的箭头，比手机地图踏实

Rokid 和高德合作了智能眼镜导航，这一点在曼谷的体感极其明显。

曼谷的巷子（Soi）编号能到几十号，街名全是泰文，门牌号毫无规律。手机导航说"往东走 200 米"——可你根本分不清东南西北。

Rokid 眼镜上的导航直接在视野里叠了一个方向箭头。你走对了，箭头是绿的；走偏了，箭头转向。就像打开了游戏里的路径指引。

其实导航这件事，回国以后我反而用得更多。

骑共享单车的时候，手机夹在车把上看不清，低头看又危险。眼镜上的箭头一直在余光里，到路口自动提示"前方左转"。你的视线始终在路上，不用在"看路"和"看手机"之间来回切。

在陌生城市，导航让你敢随便走。在熟悉的城市，导航让你可以不用想。

### 5️⃣ 不只是旅行：那些"不值得掏手机"的瞬间

曼谷回来以后，我发现这副眼镜没被放进抽屉。

因为生活里有太多"不值得掏手机，但又想看一眼"的瞬间：

*** 通知提醒** 手机揣在兜里，微信消息来了。以前你得掏出来看一眼——可能只是一条群消息，看完又塞回去。现在镜片右上角弹一行字，扫一眼就知道要不要回。

开会的时候尤其好用。手机倒扣在桌上，但重要消息不会漏——镜片上安静地闪一下，只有你自己看得见。

*** 提词器** 这个功能是意外之喜。年后公司开会要做一个英文汇报，我把讲稿传到眼镜上，对着同事讲的时候全程看着他们的眼睛。字幕在我这边滚动，他们那边什么都看不见。

讲完有人说"你英文挺流利的啊"。

不，我只是偷看了提词器。

*** 骑行导航** 周末骑车去郊区，手机塞在骑行服口袋里。以前要停下来掏手机看路线，现在箭头一直在眼前。到了岔路口，“右转，800 米后到达”。手不离车把，目不离路面。

最好的通知是"扫一眼就够了"。最好的提词是"别人觉得你没在看"。最好的导航是"你忘了自己在被引导"。

### 6️⃣ 灵珠 AI 工作流：串起旅行的一整天

灵珠平台的工作流能力，可以把零散的能力串成一条完整的旅行动线。

我搭了一个"一日行程"工作流：

```
触发 → 早上说"今天去大皇宫"
 ↓
节点1 → 智能体查天气 → 如果暴晒，追加推荐室内商场备选
 ↓
节点2 → 规划路线 → 酒店出发 → BTS + 渡船方案 → 预计到达时间
 ↓
节点3 → 推送至眼镜导航 → 出门即走
 ↓
节点4 → 到达后 → 切换为"景点讲解"模式 → 边走边介绍
 ↓
节点5 → 午饭时间 → 自动推荐附近餐厅 → 帮你看菜单
 ↓
节点6 → 下午 → 推荐周边逛街路线 → 商品识别+比价
 ↓
节点7 → 傍晚 → 推送返程路线 → 提醒 BTS 末班车时间
```

每个节点之间无缝衔接。你不需要反复唤醒、反复描述需求，工作流记得你今天要干什么。

一个人旅行最累的地方，是每一个决定都得自己做。工作流帮你接管了那些不重要的决定，把精力留给真正值得感受的东西。

### 7️⃣眼镜端 UI：少就是多

不管是翻译、导航、通知还是歌词，眼镜上的信息只遵循一个原则——余光能看清，就绝不让你转头。

翻译模式两行字：原文灰色小字，译文白色大字。 导航模式三个元素：一个箭头、一行距离、一个街道名。 通知模式一行字：来源 + 摘要，三秒后自动消失。 歌词模式两行字：原文 + 翻译，跟着节拍滚动。

```
{
 "layout": "vertical",
 "position": "bottom_center",
 "children": [
 {
 "type": "TextView",
 "props": {
 "id": "tv_original",
 "text": "สวัสดีค่ะ กี่ท่านคะ？",
 "textColor": "#AAAAAA",
 "textSize": 14
 }
 },
 {
 "type": "TextView",
 "props": {
 "id": "tv_translated",
 "text": "你好，请问几位？",
 "textColor": "#FFFFFF",
 "textSize": 20
 }
 }
 ]
}
```

旅行中每多看一秒屏幕，就少看一秒风景。UI 的责任是把自己藏好。

### 9️⃣ 它改变的，是你和世界的距离

技术参数可以列很多：89 种语言、200ms 延迟、6 种离线、49 克重量。

但真正打动我的，跟参数无关。

在曼谷，它让我第一次觉得自己"属于"一座陌生的城市。没有举着手机拍路牌的窘迫，没有对着翻译 App 面面相觑的尴尬，没有在 BTS 站台上原地转三圈的焦虑。

我走得很快，也走得很慢。快的时候像本地人一样穿过暹罗广场，慢的时候在唐人街的老药铺门口站了十分钟——因为眼镜帮我读了门口那块褪色的泰文招牌，上面写着"百年老店，祖传凉茶"。

如果没有戴眼镜，我会直接走过去。

回国以后，它安静地待在我每天出门戴的那副眼镜的位置上。骑车的时候看导航，开会的时候看通知，跑步的时候看歌词，做分享的时候看提词。

没有一个功能是离不开的。但每一个功能都让你少掏了一次手机，少低了一次头。

旅行最遗憾的事，从来都不是没去哪个景点。

是你明明在那里，却因为看不懂、听不懂，跟这座城市隔了一层玻璃。

生活最可惜的事，也从来都不是错过了什么大事。

是你每天低头看手机的那几百次，每一次都把眼前的世界按了三秒暂停。

Rokid 做的事情就是把这层玻璃擦掉，把暂停键还给你。

当语言不再是墙，每一条陌生的街道都是邀请。当屏幕不再拦在眼前，每一个当下都是完整的。
