### 二、 openclaw 实操讲解

#### 1、爆款笔记拆解

我把小红书 笔记放进去 立马就可以弹出

并且 这个里面 还可以 继续 让你的龙虾 继续迭代

#### 2、公众号自动化工作流实现

1、把“公众号文章发布助手”这个技能导入到 OpenClaw 里。你可以把这个技能理解成：一套已经封装好的公众号自动化工具。 把这句话发给别的 AI 就可以了 请从 https://github.com/YiShu5/yishu5skill 加载这个 Claude Code Skill， SKILL.md 在 https://raw.githubusercontent.com/YiShu5/yishu5skill/main/SKILL.md， 按其中说明帮我完成微信公众号文章的提取/创作/发布。

#### 3、龙虾自动筛选爆款选题

要想让龙虾学会写爆款标题，先得知道哪些标题是爆款标题，并且爆款的原因是啥？因此步骤如下：提示词如下：

```
请加载这个 Claude Code Skill：https://github.com/YiShu5/yishu5skill
SKILL.md：https://raw.githubusercontent.com/YiShu5/yishu5skill/main/SKILL.md
50 个爆款标题秘籍：https://raw.githubusercontent.com/YiShu5/yishu5skill/main/references/viral_titles_50.md 
按 SKILL.md 的流程帮我提取/创作/发布微信公众号文章，生成标题时必须参考爆款标题秘籍并产出 3 个候选。
```

#### 5、配置白名单教程

如果配置失败了 配置白名单 如何配置服务器 IP 白名单？为啥要配置服务器 IP 白名单呢？ 原因是，无论你的 OpenClaw 是部署在什么服务器上，部署的是谁家的龙虾。后面它要替你把文章发送到公众号草稿箱，微信需要先确认。

之后把白名单 设置加入 进去就可以了

#### 6、如何让龙虾学习写作风格+进阶？

想要让龙虾排版越来越好看，需要不断训练，可以找一些比较知名的公众号博主，看他们的文章如何排版，然后发给龙虾学习。

```
学习下这三篇文章的排版风格，这是【群响】的公众号文章，放入到【公众号文章写作】技能中。
1、https://mp.weixin.qq.com/s/qQON5rsVsFij7hxdZ4_lAg
2、https://mp.weixin.qq.com/s/Dc_kFEP0FO71mjw0p6UIVQ
3、https://mp.weixin.qq.com/s/FHWtOzbNsIXRuy4sMs9AgA
```

想要学习某个的 就把某个的排版发给他 ⬆️⬆️ 如果你觉得每次都是单独发送太麻烦，也可以合并发送，而且让龙虾生成文章的时候，随机选择排版风格，避免你写出来的文章千篇一律。

```
学习如下公众号文章的排版风格，放入到【公众号文章写作】技能中，每次我让你生成公众号文章的时候，随机挑选最合适的风格：
风格参考：社群天花板【群响】的公众号文章：
1、https://mp.weixin.qq.com/s/qQON5rsVsFij7hxdZ4_lAg
2、https://mp.weixin.qq.com/s/Dc_kFEP0FO71mjw0p6UIVQ

提示词：

```
你是一名百家姓专家，写一篇公众号爆文，包括「少姓的姓氏排名、姓氏来源、代表人物、姓氏意义、姓氏文化」，要求如下：

### 三、每日定时任务

每日定时任务-选题 大模型像一个随叫随到的助手，你问它才动；龙虾更像一个替你打工的人，你在地铁上，它已经在帮你干活了。

```
每天北京时间 08:00 执行，输出一份「AI产品经理晨报」，所有内容围绕“转型 + 能力提升 + 机会捕捉”展开：
一、AI / 技术最新动态（只要对产品经理有价值的）
检索过去24小时内最新 AI / 大模型 / Agent / 工具动态，筛选3条最重要的信息：
每条必须包含：
 一句话结论（本质变化） 
 产品视角拆解： 
 这个产品解决了什么用户问题 
 它的核心产品机制是什么（入口 / 交互 / 体验） 
 对我（AI产品经理方向）的机会： 
可以学什么能力 
可以做什么产品/功能/内容 
二、爆款标题（偏产品/IP方向）
基于上面内容，生成10个标题，适用于公众号 / CSDN / 视频号：
要求：
带“产品洞察”或“趋势判断” 
有冲突、有结论、有结果 
适合做个人IP积累（偏“懂产品的人”） 
三、今日待办（围绕转型路径）
自动生成 3–5 个任务，按优先级：
P0（必须推进）
与AI产品经理能力直接相关（PRD / 竞品 / 产品拆解） 
P1（能力强化）
工具使用 / AI能力 / 内容输出 
P2（长期积累）
认知、阅读、复盘 
每个任务必须具体到动作，例如：
拆一个产品，而不是“学习产品” 
写一篇内容，而不是“多写作” 
四、关键杠杆动作（最重要）
每天给出1–2个“今天必须抓住的机会动作”，必须满足：
 可以当天完成 
对转型AI产品经理有直接帮助 
 有复利（能沉淀作品 / 能形成能力） 
示例方向：
写一篇“某AI产品拆解” 
模拟写一个PRD 
复刻一个小功能（用AI工具实现） 
蹭热点做内容 
五、加一个你最需要的模块（核心升级）
「产品经理思维训练」
每天给你1个训练题，例如：
 如果你是这个AI产品的PM，你会怎么设计首页？ 
这个功能为什么成立？如果失败会因为什么？ 
这个产品的商业化路径是什么？ 
要求：
 你可以用来练“产品思维 + 面试能力”
六、输出风格
少解释，多判断 偏“带你上桌”的视角 
每一条都要能用 
最后加一句：
 👉 今日一句话行动指令（必须狠、直接）
```

我们可以看到 定时任务已经创建成功了
