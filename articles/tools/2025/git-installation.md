### 前言

Git是一个免费的开源分布式版本控制系统。它易于学习，占用空间小且性能极快。

#### 最新版本

- 版本：2.50.0
- 发布说明：(2025-06-16)

#### 安装步骤

Git 进入官网 点击红框下载 点击Next。 这时候请将Git安装在目录名称不含中文和空格的路径下，然后点击Next。 点击Next。 这里是默认编辑器 选择VIM就可以了 设置分支名，点击Next。 修改Git的环境变量，选第一个，不修改环境变量，只在 Git Bash 里使用 Git。 请在选择后台客户端连接协议时，采用默认的OpenSSL选项，点击Next。 配置文末换行符，点击Next。 选择 Git 终端类型，选择默认的 Git Bash 终端，，点击Next。 选择 Git pull 合并的模式，选择默认，点击Next。 选择Windows的凭据管理器 其他配置，选择默认设置，点击安装就可以了。 安装完成 点击finish

#### 安装完毕切到电脑桌面

右键鼠标会看到下图出现两个：Git GUI Here和Git Bash Here

- Git GUI Here是可视化操作工具
- Git Bash Here是Git配套的控制台

点击这两个进行打开，没问题了就可以关闭了

在 Git Bash 终端里输入 git --version 查看 git 版本，如图所示，说明 Git 安装成功。
